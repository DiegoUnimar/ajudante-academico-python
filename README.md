# Assistente Acadêmico

Projeto desenvolvido em Python com o objetivo de auxiliar estudantes na organização da rotina acadêmica.

## Funcionalidades

- Cadastro de alunos
- Organização de horários de aula
- Registro de datas de provas
- Armazenamento de dados em arquivos JSON

## Tecnologias Utilizadas

- Python
- JSON
- Estruturas de dados (listas e dicionários)

## Objetivo

Praticar conceitos de programação, manipulação de arquivos e desenvolvimento de aplicações em linha de comando.

## Como Executar

1. Clone o repositório
2. Abra o projeto em um ambiente Python
3. Execute o arquivo principal
